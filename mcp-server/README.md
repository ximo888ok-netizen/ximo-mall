# Ximo Mall MCP Server

MCP (Model Context Protocol) 服务器，将 Ximo Mall AI 电商详情页生成平台的全部能力封装为标准 MCP 工具，供 AI 客户端（如 Claude Desktop、Cursor、VS Code 等）调用。

## 快速开始

### 启动服务器

```bash
# HTTP 模式（推荐，含 Web UI）
cd mcp-server
npm run start:http

# 或者 Stdio 模式（适用于 Claude Desktop）
npm run start:stdio
```

HTTP 模式下默认监听 http://localhost:3001。

### 配置环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `XIMO_MALL_API_URL` | Ximo Mall 应用地址 | `http://localhost:3000` |
| `PORT` | MCP 服务器端口 | `3001` |

## 集成到 AI 客户端

### Claude Desktop

在 `claude_desktop_config.json` 中添加：

```json
{
  "mcpServers": {
    "ximo-mall": {
      "command": "npx",
      "args": ["tsx", "path/to/mcp-server/src/index.ts"],
      "env": {
        "XIMO_MALL_API_URL": "http://localhost:3000"
      }
    }
  }
}
```

### Cursor

在 Cursor 设置中配置 MCP 服务器，指向 stdio 入口。

### VS Code (GitHub Copilot)

在 VS Code 中配置 MCP 服务器指向 stdio 入口。

## Web UI

启动 HTTP 模式后，访问 http://localhost:3001 即可打开管理界面，可在浏览器中直接测试所有工具。

功能：
- 📋 按分类浏览所有 63 个工具
- 🔍 搜索工具
- ✏️ 编辑请求参数
- ▶️ 直接调用 MCP 工具
- 📊 查看响应结果

## 技术架构

```
AI 客户端 (Claude/Cursor/等) 
        │
        ▼  MCP 协议 (HTTP/stdio)
        │
┌─────────────────────────────┐
│     Ximo Mall MCP Server  │
│                             │
│  ┌───────────────────────┐  │
│  │   MCP 服务器 (McpServer) │  │
│  │   - 63 个 Tools       │  │
│  └────────┬──────────────┘  │
│           │ HTTP Client     │
│  ┌────────▼──────────────┐  │
│  │   API Client          │  │
│  │   (调用 Next.js API)  │  │
│  └───────────────────────┘  │
└─────────────────────────────┘
        │
        ▼  HTTP
        │
┌─────────────────────────────┐
│    Ximo Mall Web App      │
│  (Next.js API Routes)       │
└─────────────────────────────┘
```
